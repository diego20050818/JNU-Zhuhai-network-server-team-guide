# windows有线网连接
> 请在安装前退出“360安全卫士”及“360杀毒”，以保证正常安装
完成安装后，请重启电脑

## 1. 打开设备管理器
![alt text](img/connect09.png)

## 2. 选择网络适配器，查看网卡
记住你的网卡是啥~
查找拥有以下字段的项：
- Ethernet(以太网)
- GbE(千兆以太网)

大多都是以下两种网卡适配器：
1. **Realtek PCle GbE Family Controller**
2. **Intel(R) Ethernet Connection (3)1218-V**

![alt text](img/connect10.png)
